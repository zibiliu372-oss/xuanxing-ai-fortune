# 玄星 AI 算命系统

一个可直接打开的前端原型，包含中式命理和西式占星两条入口，并为每条路线设计了动态版 UI 和静态版 UI。

## 打开方式

直接用浏览器打开：

```text
/Users/daniu/Desktop/人物/ai-fortune-system/index.html
```

## 当前能力

- 入口选择：中式命理、西式占星
- UI 版本：动态版、静态版
- 输入项：姓名、出生日期、出生时间、出生地、主题、具体问题
- 本地生成：根据输入生成娱乐向 AI 风格解读
- 报告展示：中式八字/五行报告，西式太阳/月亮/上升报告

## 接真实 AI 的位置

当前解读由 `app.js` 里的 `buildReading()` 本地生成。后续如果要接 OpenAI 或自有后端，可以把这个函数替换成接口请求，并保留现有 UI 不变。
